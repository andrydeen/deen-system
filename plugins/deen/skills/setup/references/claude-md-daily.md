This folder is the operator's primary daily journal. One note per day.

## When To Write Here

Whenever meaningful work happens (not casual chat), append a session log to `Daily/YYYY-MM-DD.md`. Create the file if it doesn't exist yet for today.

## Daily Note Frontmatter

```yaml
---
type: daily-note
date: YYYY-MM-DD
---
```

## Shape

Keep entries short and specific. A session block per working session:

```markdown
## Session
- **Focus**: [what you worked on]
- **Completed**: [what got done]
- **Next Steps**: [what's next]
```
